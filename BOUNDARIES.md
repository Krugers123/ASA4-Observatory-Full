# Disclosure Boundaries

This repository is a public-safe description of ASA4 Observatory Full Edition.

It is not the private runtime repository.

## Included

This repository may include:

- public-safe architecture summaries
- module descriptions
- screenshots of the private full dashboard
- capability framing
- release boundaries
- research/disclaimer notes

## Not Included

This repository does not include:

- private source code
- default credentials
- scoring internals
- hidden thresholds
- proprietary calibration logic
- private datasets
- raw private conversation archives
- deployment secrets
- production security configuration
- sensitive operator workflows

## Important Boundary

Screenshots are used to show the operator-facing shape of the system.

They should not be interpreted as a public deployment manual, a credential guide, or a disclosure of the private implementation.

## Correct Public Reading

ASA4 Observatory Full Edition should be understood as:

- a private research/operator build
- a full dashboard layer for trajectory observability
- a larger internal continuation of the public ASA Observatory line
- a system still undergoing validation before broader release
